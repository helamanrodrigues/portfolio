Notes to the Idaho Town Information

Helaman Rodrigues Final Project

Description of my website:

This website was developed to show some cities that have in Idaho, located in the United States, where you can see some informations about the city like the weather, events, articles and galleries.

The objective idea is to know more about the city, the weather that have there and to advice others about the situation of the weather and if have some storm there.

CSS Features used:

  • Media query: To make the page responsive.
  • Hover: To change the color of the search button.
  • Box-sizing + moz-box-sizing + webkit-box-sizing: apply a natural box layout model to all elements
  • Clearfix: Force an element not to collapse
  • Root: To make easier and organized to select the colors
  • Lazy Load: Strategy to identify resources as non-blocking (non-critical) and load these only when needed.
  • Normalize: Make browsers render all elements more consistently and in line with modern standards.

Techonologies that are used:

  • HTML
  • CSS
  • JavaScript
  • API
