Notes to the NomadsTrip App

Helaman Rodrigues Final Project

Description of my App:

My TravelApp is a dynamic website-application that you can fill the form with your informations like city, country, zip code, name, birth date and others info to check the location that you are looking for. The app provides you to save information of someplace in the world that you want to know or visit by google maps and to plan for visit there someday.

So this are the steps of the app:

  1. Click on New Button to add the new user
  2. Enter your name
  3. Enter your last name
  4. Enter your company
  5. Enter your address
  6. Enter your city
  7. Enter your state
  8. Enter your zip code
  9. Enter your country
  10. Enter your phone
  11. Enter your birth date
  12. Save your info
  13. Go down the form to check the map of the place that you selected and the weather condition of this place
  14. You can Edit the info, Clear the info or create a New one.
  15. If you want to delete the user that you created, just select the X button

The objective idea is to be able to have a special information that will help you decide and learn more about the place that you want to go.

CSS Features used:

  • Media query: To make the page responsive.
  • Hover: To change the color of the search button.
  • Box-sizing + moz-box-sizing + webkit-box-sizing: apply a natural box layout model to all elements
  • Clearfix: Force an element not to collapse
  • Root: To make easier and organized to select the colors

What Problems did you have and how did you resolve them?

My main problem was to get the api of booking travel, but I can't pay for that, so I decided to change my project using the google maps.

Other problem that I had was to make the layout responsible and different for mobile, desktop or in other words in the small and medium/large css. I fixed them checking my others apps that I developed before and trying to put the same conditions there.

The other point was the javascript to make work the interaction with the final user, one of my friends give me a help with that.

How did you use the coding topics from the course?

I could say that in my final project I saw the most important topics such as: creating dynamic pages, using external information through APIs, good handling of CSS, etc.
